# TokenFlow

A Web3 platform to create and manage token infrastructure.

## Features
- ERC20 Token Creation
- Token Vesting (Time-lock)
- Staking System

## Tech Stack
- Solidity (Hardhat)
- React
- Ethers.js

## 🚀 Live Demo

- Token deployed on Sepolia
- Staking + Vesting working

## 🧪 Features

✔ ERC20 Token  
✔ Vesting Contract  
✔ Staking with Locking  
✔ Reward System
